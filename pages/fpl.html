Premier League Player Value Predictor

Analysis Overview

This MVP prediction model uses an XGBoost machine learning algorithm to analyze 15 performance metrics (goals, assists, clean sheets, saves, influence, creativity, threat, and more) from Premier League players. The predictions are based on Fantasy Premier League's official scoring system, which calculates each player's value_season based on their on-field performance throughout the season.

The model processes these metrics through gradient boosting to identify patterns and predict player value, displaying the results in the interactive dashboard below. Data updates automatically as new match statistics become available.
